# Setup electron program with yeoman
see http://qiita.com/makoto_kw/items/fb419d2c35033a17ebb4

## Preparation
```sh
npm install -g yo bower grunt-cli gulp generator-electron
```

## Create package
```sh
mkdir foo
cd foo
yo electron
```

## run ap
```
npm start
```

## packaging
```
npm run build
```

## testing
```
npm test
```
